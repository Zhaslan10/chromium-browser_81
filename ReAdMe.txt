Помещаем в папку "EUBANK" папку "chromium" содержащую необходимые файлы, 
которые можно выкачать из http://security.ubuntu.com/ubuntu/pool/universe/c/chromium-browser/
у меня это 3 файла (chromium-browser_*, chromium-codecs-ffmpeg-extra_*, chromium-browser-l10n_*) 
для Вашего дистрибутива  
 
chromium.sh - скрипт установки.

команда запуска скрипта 
sudo /bin/bash /home/techman/EUBANK/chromium/chromium.sh
